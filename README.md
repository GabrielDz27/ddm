# Desenvolvimento de Dispositivos Móveis

Repositório dedicado ao armazenamento dos projetos e práticas desenvolvidos na disciplina de Desenvolvimento de Dispositivos Móveis, sob a orientação do Professor Rômulo.

## Estrutura do Repositório

> **Nota:** Para facilitar a organização, cada aplicativo foi desenvolvido em uma branch separada. Navegue pelas branches do repositório para acessar o código-fonte de cada projeto específico.

---

## Aplicativos Desenvolvidos

### 1. Contador de Cliques
* **Objetivo:** Desenvolvimento de um aplicativo simples para contabilizar cliques do usuário.
* **Conceitos Aplicados:** Manipulação de componentes visuais, eventos de interface e atualização dinâmica de dados na tela.

### 2. Sorteio de Números Aleatórios
* **Objetivo:** Geração de números aleatórios dentro de um intervalo definido pelo usuário.
* **Conceitos Aplicados:** Entrada de dados, validações e exibição de resultados em tempo real.

### 3. Slide Show
* **Objetivo:** Apresentação sequencial de imagens.
* **Conceitos Aplicados:** Navegação entre telas/componentes, controle de imagens e interação com botões.

### 4. Calculadora de IMC (Uma Tela)
* **Objetivo:** Cálculo de Índice de Massa Corporal com interface unificada.
* **Conceitos Aplicados:** Captura de peso/altura, lógica de cálculo no clique do botão e exibição do perfil do usuário com base em imagens dinâmicas.

### 5. Calculadora de IMC (Duas Telas)
* **Objetivo:** Mesma lógica da Calculadora de IMC, porém distribuída em duas telas.
* **Conceitos Aplicados:** Navegação entre Activities, passagem de dados através de `Bundle` e renderização de resultados na segunda tela.

### 6. Lista de ViewGroups
* **Objetivo:** Prática focada na estruturação de layouts.
* **Conceitos Aplicados:** Exploração a fundo dos mecanismos de posicionamento e alinhamento de componentes gráficos no ecossistema Android.

### 7. Simple Paint
* **Objetivo:** Criação de uma aplicação de desenho livre na tela.
* **Conceitos Aplicados:** Criação de `View` personalizada, manipulação de `Canvas` e `Paint`, e tratamento de eventos de toque (`MotionEvent`).
